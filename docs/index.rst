Errbot Discord Backend Documentation
========================================================================

Welcome to the ``err-backend-discord`` documentation page.  You'll be able to find
installation instructions, configuration information and examples of using some of the backend's features.

The ``err-backend-discord`` backend lets you connect to the `Discord <https://discord.com/>`_ platform. `A place that makes it easy to talk every day and hang out more often.`
